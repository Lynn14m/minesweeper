RUNNING THE APPLICATION:

Pre-requisites:
    - Python 3.8.5 (https://www.python.org/downloads/release/python-385/)
    - Pip (https://pypi.org/project/pip/) ...sometimes already installed with Python
    - Ideally Chrome or Firefox

Step 1.
    From the project root run the following command (depending on your python installation):
        pip3 install -r requirements.txt
        OR
        pip install -r requirements.txt

Step 2.
    From the project root run the following command (depending on your python installation):
        python3 manage.py runserver
        OR
        python manage.py runserver

Step 3.
    Navigate to http://localhost:8000 or http://127.0.0.1:8000

Step 4.
    Play Minesweeper!
